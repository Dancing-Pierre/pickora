Before editing frontend code, read:

1. [Directory Structure](./directory-structure.md)
2. [Type Safety](./type-safety.md)
3. [Quality Guidelines](./quality-guidelines.md)
4. [Component Guidelines](./component-guidelines.md) for UI/template/style changes
5. [Hook Guidelines](./hook-guidelines.md) for composables, API wrappers, or reusable stateful logic
6. [State Management](./state-management.md) for local state, history, draw flow, or persistence changes
